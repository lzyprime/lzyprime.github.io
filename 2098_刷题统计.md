## code

- 一周刷题量 `week = 5 * a + 2 * b`, 可以先算出整周刷题刷了几周：`n / week`
- 剩余 `n % k` 道题，不足一周。
- `res`记录需要的天数, `res % 7` 判断当前星期几, `(res % 7 == 6 || res % 7 == 0)` 为双休日

### c++

```cpp
#include <bits/stdc++.h>
using namespace std;
int main() {
    long long a, b, n;
    cin >> a >> b >> n;
    long long week = 5 * a + 2 * b, res = (n / week) * 7;
    n %= week;
    while (n > 0) {
        res++;
        n -= (res % 7 == 6 || res % 7 == 0) ? b : a;
    }
    cout << res << endl;
    return 0;
}
```
